# Akari Math Lab (Vercel + microCMS)

このリポジトリは Next.js 製のフロントエンド（`akari-vercel/` ディレクトリ配下）だけを残し、Vercel へのデプロイに必要な構成に整理しました。開発・デプロイ手順は `akari-vercel/README.md` を参照してください。

## クイックスタート

```bash
cd akari-vercel
npm install
npm run dev
```

`MICROCMS_SERVICE_DOMAIN` と `MICROCMS_API_KEY` などの環境変数も `akari-vercel/.env.example` を基に設定してください。
