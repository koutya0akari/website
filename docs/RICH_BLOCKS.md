# 折り畳み・タブ・隠しの使い方

日記 / 数学メモ / メモの本文（Body）で、`:::` 記法を使って
**折り畳み**・**タブ**・**隠し（スポイラー）** を簡単に書けます。

エディターのツールバーの **「レイヤー」アイコン（Layers）** からも雛形を挿入できます。
内側には通常どおり Markdown・数式・画像・リンクをそのまま書けます。

## 折り畳み（fold）

クリックで開閉できるブロック。最初は閉じています。

```text
:::fold 証明を見る
**ポイント**: $a^2 + b^2 \geq 2ab$。

- 箇条書きも使えます
:::
```

`:::fold` の後ろがタイトルになります。省略すると「詳細」になります。

## タブ（tabs）

`@tab ラベル` で区切ると、ラベルごとに切り替えられるタブになります。

```text
:::tabs
@tab Python
\`\`\`py
print("hello")
\`\`\`
@tab 説明
これは *説明* タブです。数式 $e^{i\pi} = -1$ も書けます。
:::
```

## 隠し（hide / スポイラー）

### ブロック（複数行）

```text
:::hide
ネタバレ注意。クリックすると表示されます。
:::
```

`:::hide` の後ろにタイトルを書くこともできます（省略すると「クリックして表示」）。

### インライン（1 行の一部）

文中の一部だけを隠したいときは `||` で囲みます（Discord 風）。

```text
犯人は ||執事|| でした。
```

ホバー（PC）またはタップ（スマホ）で表示されます。

## 補足

- これらの記法はコードブロック（``` … ```）の中では展開されません。
- 旧来の数学ブロック（`math-callout`）については [MATH_BLOCKS.md](./MATH_BLOCKS.md) を参照してください。
